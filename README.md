# Python-Data-Analysis-Visualization-Projects

# 📊 Python Data Analysis & Visualization Projects

## 📌 Project Overview

This repository contains hands-on practice and mini-projects using Python libraries such as NumPy, Pandas, and Matplotlib. The goal of this project is to build a strong foundation in data analysis, data manipulation, and data visualization.

---

## 🛠️ Technologies & Tools

* Python
* NumPy
* Pandas
* Matplotlib
* Jupyter Notebook / VS Code

---

## 📂 Project Contents

### 🔢 NumPy Assignment

* Created 1D and 2D arrays
* Performed statistical operations (mean, median, standard deviation)
* Array reshaping and slicing
* Matrix operations and dot product

---

### 📊 Pandas Assignment

* Created DataFrames from structured data
* Performed data filtering and grouping
* Calculated summary statistics
* Added new columns and sorted data
* Exported data to CSV file

---

### 📈 Matplotlib Assignment

* Line plots for trend analysis
* Bar charts for comparison
* Pie charts for distribution
* Histograms for frequency distribution

---

## 🎯 Learning Outcomes

* Data manipulation using Pandas
* Numerical computation using NumPy
* Data visualization using Matplotlib
* Understanding of real-world data analysis workflow
